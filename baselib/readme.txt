安装说明： 

1. 目前只有群共享的exe安装包才会自动安装插件,  插件目录： "C:\Program Files\FTNN\Plugin\"

2. 使用文档请参见git : https://github.com/FutunnOpen/OpenQuant/开放接口/Python接口

3. config.ini 默认开启本地监听, 如要开启远程监听，请修改ip=0.0.0.0,  ip/port修改后， 注意同步更改python脚本中的参数

4. 腾讯云上部署了体验环境可以访问行情接口:  ip =119.29.141.202,  port = 11111 , 可不用安装客户端直接体验
